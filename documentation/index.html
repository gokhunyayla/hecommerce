Introduction

HeCommerce: Revolutionizing the E-Commerce Experience with Our Advanced HTML Template

Welcome to the future of e-commerce with HeCommerce! This innovative HTML template brings together everything a modern e-commerce site needs. HeCommerce isn’t just a template; it’s a key to unlocking your business’s potential.

Versatile Product and Page Listing Options: HeCommerce offers the flexibility to showcase your products and content in both grid and list formats. This empowers your visitors to customize their shopping experience to suit their preferences.

Additional Page Structures and Standard Pages: Beyond the standard e-commerce pages, HeCommerce stands out with its additional page structures. These extra pages highlight your brand’s uniqueness and professionalism.

User-Friendly Cart Features: The shopping cart is an essential element of any e-commerce site, and HeCommerce excels in this area. Our intuitive and seamless cart features ensure a smooth and hassle-free shopping experience for your customers.

Responsive Design for All Devices: Understanding the importance of mobile commerce, HeCommerce is completely responsive. This ensures that your site looks great and functions flawlessly on all devices, from desktops to smartphones.

Instant Visibility of Customer Reviews: One of the standout features of HeCommerce is the immediate visibility of customer reviews on the product page. As soon as the page loads, reviews are prominently displayed at the top-right corner, instilling trust and transparency.

Getting Started

Unzip the archive and find out the folder html. There are all the template files in this folder.

The file and folder structure is as below;

Assets

  1. assets/css - folder with template css files
  2. assets/fonts - folder with font files
  3. assets/images - folder with images files
  4. assets/js - folder with template js files
  5. assets/vendor - folder with vendor files as Bootstrap 5, Owl Carousel

Upload the template files to server with the help of one of the FTP-clients like FileZilla.

Media Files

  1. media/product - folder with product images
  2. media/campaign - folder with campaign images
  3. media/page - folder with page images
  4. media/slider - folder with slider images

Head CSS Structure

Followings are the css files which loaded inside the Head Section:

                            
<link rel="icon" href="assets/images/favicon.png" type="image/png"/>
<link rel="stylesheet" href="assets/vendor/bootsrap/css/bootstrap.min.css"/>
<link rel="stylesheet" href="assets/vendor/bootstrap-icons/font/bootstrap-icons.css"/>
<link rel="stylesheet" href="assets/vendor/owl-carousel/assets/owl.carousel.min.css">
<link rel="stylesheet" href="assets/vendor/owl-carousel/assets/owl.theme.default.css">
<link rel="stylesheet" href="assets/css/style.css" type="text/css"/>

Javascript Structure

Followings are the js files which loaded bottom of the Body Section:

                            
<script src="assets/vendor/bootsrap/js/bootstrap.bundle.js"></script>
<script src="assets/vendor/jquery/jquery-3.7.1.min.js"></script>
<script src="assets/vendor/owl-carousel/owl.carousel.min.js"></script>
<script src="assets/js/script.js"></script>

Fonts

Google's Free Roboto font is used as a default font family for the template and the related files are inserted to the head section as below;

                            
<link rel="stylesheet" href="assets/fonts/Roboto/roboto-fonts.css"/>

Owl Carousel

Below script's are used the initialize owl carousels.

                            
//Carousel for Slider and Header's
$('.owl-slider').owlCarousel({
    loop:true,
    margin:10,
    nav:true,
    responsive:{
        0:{
            items:1
        },
        600:{
            items:1
        },
        1000:{
            items:1
        }
    }
})

//Carousel of Bottom of the product pages for Related Products
$('.owl-related-products').owlCarousel({
    loop:true,
    margin:10,
    nav:true,
    responsive:{
        0:{
            items:1
        },
        600:{
            items:3
        },
        1000:{
            items:6
        }
    }
})

//Carousel for 3 products visible for lg and xl sizes
$('.owl-trial').owlCarousel({
    loop:true,
    margin:20,
    nav:true,
    responsive:{
        0:{
            items:1
        },
        600:{
            items:2
        },

            items:2
        },
        1200:{
            items:3
        }
    }
})

Google Map & How to add a Google Map to The Website

1. Go to https://maps.google.com and Type the location where you want to add to the website. Then click on "Share" icon button.

2. Click "Embed a map" tab and then click to "COPY HTML" text button.

3. Paste the copied link into the html file on contact.html page.
